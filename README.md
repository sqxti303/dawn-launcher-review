## Dawn Launcher 是什么，能干嘛

简单说：按个快捷键或者鼠标中键，一个分好类的启动面板弹出来，点一下就开软件、打开文件、跳到网站。关掉面板，桌面还是干干净净的。

类似的思路早年有个叫"音速启动"的老软件，用过的人应该有印象。Dawn Launcher 就是参考了这个思路，用现代技术重新做了一遍，而且做到了 2.x 版本。

**核心功能大致这些：**

- **一键速启**：软件、文件、文件夹、网站，加进去就能一键打开
- **分组管理**：按工作/娱乐/设计/开发随意分类，子分类也支持
- **关联文件夹**：把常用目录关联进来，文件夹内容实时同步展示，不用再一层层点开资源管理器
- **快速搜索**：输入几个字就能定位到目标项目，支持直接发起网络搜索
- **高度自定义**：主题、背景图、字体、图标大小，想折腾就折腾

到了 Dawn Launcher 2，又多了一些更实用的东西：分类可以独立开窗、可以设密码、支持自动备份数据、可以设环境变量——这最后一条对程序员来说挺有用，相当于给 Dawn Launcher 配了一套专属的环境变量管理。
